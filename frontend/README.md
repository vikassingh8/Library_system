# Library System - Frontend

React frontend for the Online Library Management System.

## Requirements

- Node.js 18+
- yarn or npm

## Setup

1. Go to frontend folder
```bash
cd library-system/frontend
```

2. Install dependencies
```bash
yarn install
```

3. Run the dev server
```bash
yarn dev
```

Frontend runs at `http://localhost:5173`. Make sure the backend is running at `http://localhost:8000`.
